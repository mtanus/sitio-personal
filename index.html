Análisis histórico de redes móviles e Internet

Dashboard interactivo con Looker Studio, para visualizar resultados del análisis de diversas fuentes de datos históricas de redes móviles e Internet.

Como trabajo final de mi formación en Big Data realicé un análisis de diversas fuentes de datos históricas de redes móviles e Internet. El proceso requirió las siguientes etapas:

  • Recolección de datos de diversas fuentes
  • Transformación y limpieza de datos utilizando Google Sheets
  • Generación de un dashboard interactivo para visualizar resultados con Looker Studio
